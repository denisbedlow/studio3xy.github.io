## Overview

Upload a **CMX3600 EDL** file and convert it to spreadsheet-friendly formats or a cleaned-up EDL. All processing happens in your browser — no files are uploaded to any server.

| Feature | Details |
|---|---|
| **Input** | CMX3600 EDL (.edl) |
| **Output** | CSV, TSV, JSON, cleaned EDL |
| **Timecode** | SMPTE, milliseconds, total frames, seconds — with framerate conversion |
| **Processing** | Remove duplicates, filter BLACK edits, sort by record TC, offset timecode |
